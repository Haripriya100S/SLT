# Sign Language Translator
Sign language is a rich and expressive form of communication used by the Deaf and hard of hearing community worldwide. Our Sign Language Translator app is a groundbreaking tool designed to empower communication and promote inclusivity. The system aims to break down communication barriers, ensuring that Deaf and hard of hearing individuals have equal access to information and communication. Our system recognizes and translates various sign languages using the trained lstm model, making it versatile and adaptable to diverse cultural and linguistic contexts.
